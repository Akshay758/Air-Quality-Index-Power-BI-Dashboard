# Air Quality Index (AQI) Analysis – Power BI Dashboard

## Project Overview
This project focuses on analyzing Air Quality Index (AQI) data using Power BI to understand air pollution levels and environmental conditions across different locations. The dashboard provides interactive visualizations that help identify pollution trends, compare AQI levels, and analyze the impact of different pollutants on air quality.

## Objective
The main objective of this project is to analyze air quality data and create an interactive dashboard that helps users easily understand pollution patterns and environmental conditions.

## Dataset Features
The dataset contains information about air pollution and environmental factors, including:

- **City** – Location where air quality was measured  
- **Date** – Date of AQI measurement  
- **AQI Value** – Overall air quality index value  
- **AQI Category** – Air quality category (Good, Moderate, Poor, etc.)  
- **PM2.5** – Fine particulate matter concentration  
- **PM10** – Particulate matter concentration  
- **NO2** – Nitrogen dioxide level  
- **SO2** – Sulfur dioxide level  
- **CO** – Carbon monoxide level  
- **O3** – Ozone level  

## Tools & Technologies Used
- **Power BI**
- **Data Cleaning and Transformation (Power Query)**

## Dashboard Features
The Power BI dashboard includes:

- AQI trend analysis over time
- Comparison of pollution levels across cities
- Visualization of major pollutants (PM2.5, PM10, NO2, SO2, CO, O3)
- AQI category distribution
- Interactive filters and slicers for better analysis

## Key Insights
- Identification of cities with the highest pollution levels
- Analysis of pollutant contribution to AQI
- Understanding air quality trends over different time periods

## Conclusion
This project demonstrates how Power BI can be used to analyze environmental data and create interactive dashboards. The insights generated from this analysis can help in understanding pollution patterns and support environmental monitoring and decision-making.
